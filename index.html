Frequently Asked Questions

What are hidden events on Luma?

Hidden events are public events on lu.ma that don't appear on Luma's featured discover page. They exist in organizer calendars and community calendars but Luma's algorithm doesn't surface them. We find all events and list them here — including tech events, AI events, startup events, and event dinners you'd otherwise miss.

Which cities do you cover?

We cover events in Bangalore (Bengaluru), San Francisco (SF/SFO), Singapore, Mumbai, New Delhi, Boston (MA), and Pune. We find tech events, AI events, startup events, crypto events, and community events in each city.

How do I find events near me?

Select your city from the navigation bar to see all events — both featured and hidden. Use the search box to filter by event type (tech, AI, startup, crypto, networking). Every event links directly to its Luma page where you can RSVP.

Is this affiliated with Luma?

No. This is an independent tool to help you find events. All events are hosted on luma.com and link directly to their Luma pages.
